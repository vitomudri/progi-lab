# Platforma Kuhari

## Opis projekta

Ovaj projekt je rezultat timskog rada u sklopu projeknog zadatka kolegija [Programsko inženjerstvo](https://www.fer.unizg.hr/predmet/proinz) na Fakultetu elektrotehnike i računarstva Sveučilišta u Zagrebu. 

Platforma kuhari je interaktivna platforma za povezivanje instruktora kuhanja s polaznicima njihovih programa. Ona omogućuje profesionalno i strukturirano učenje i komunikaciju. Neki od problema koje rješava su nedostatak dobro organizirane komunikacije, nedostatak strukture i hijerarhije, teško praćenje napretka, recepti raspršeni po raznim lokacijama i slično.
